# 3D Interactive Portfolio

A modern, immersive developer portfolio built with React, Three.js, and Framer Motion. This project showcases technical skills and projects through a visually stunning 3D interface with glassmorphism design elements.

## üöÄ Features

*   **3D Interactive Elements**: Immersive background scenes using Three.js and React Three Fiber.
*   **Glassmorphism UI**: Modern, sleek interface with frosted glass effects and dynamic gradients.
*   **Responsive Design**: Fully optimized for desktops, tablets, and mobile devices.
*   **Dynamic Navigation**: Smooth scrolling and active section highlighting.
*   **Project Showcase**: Detailed project cards with tech stacks and links.
*   **Skills Visualization**: Interactive display of technical and professional skills.
*   **Experience Timeline**: A structured view of professional history.
*   **Web Resume**: A dedicated, printable resume page (`/resume`) that dynamically generates a professional CV from portfolio data.
*   **Admin Console**: (Implied/Planned) Interface for managing portfolio data.

## üì¶ Installation

1.  Clone the repository:
    ```bash
    git clone https://github.com/ShakthiRithanya/Portfolio-Website.git
    ```
2.  Navigate to the project directory:
    ```bash
    cd Portfolio-Website
    ```
3.  Install dependencies:
    ```bash
    npm install
    ```
4.  Start the development server:
    ```bash
    npm run dev
    ```
